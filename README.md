Sistema de Técnicas Bancárias Interativo
Visão Geral
Este projeto apresenta um sistema interativo desenvolvido em Python utilizando a biblioteca ipywidgets para catalogar e fornecer descrições detalhadas de diversas técnicas e operações bancárias. O objetivo é criar uma ferramenta acessível e fácil de usar para explorar conceitos financeiros e suas aplicações práticas no setor bancário.

Funcionalidades
Interface Interativa: Utiliza um menu suspenso (Dropdown) para permitir que o usuário selecione facilmente diferentes técnicas bancárias.
Descrições Dinâmicas: Exibe instantaneamente descrições claras e concisas para cada técnica selecionada, fornecendo um entendimento rápido do seu propósito e funcionalidade.
Abrangência de Tópicos: Cobre uma variedade de áreas, incluindo:
Cálculo de juros em planilhas Excel.
Gestão de dados de clientes e limites de crédito.
Consultas SQL para saldo e histórico de transações.
Simulação de parcelas de empréstimos.
Relatórios de gastos e compras no cartão.
Análise de empréstimos em aberto.
Como Usar
Para executar e interagir com este sistema, siga os passos abaixo:

Abra o Notebook: Faça o download do arquivo .ipynb ou abra-o diretamente no Google Colab ou em qualquer ambiente Jupyter compatível.
Execute as Células de Código: Execute todas as células do notebook. A interface interativa será exibida abaixo da última célula de código.
Selecione uma Técnica: Utilize o menu suspenso rotulado "Técnica:" para escolher uma das opções disponíveis.
Visualize a Descrição: A descrição correspondente à técnica selecionada será exibida na área de output logo abaixo do menu suspenso.
Tecnologias Utilizadas
Python: Linguagem de programação principal.
ipywidgets: Para criar a interface de usuário interativa.
IPython.display: Para controlar a exibição de elementos interativos no ambiente Jupyter/Colab.
